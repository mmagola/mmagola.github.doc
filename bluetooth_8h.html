Detailed Description

Bluetooth library for KL46Z and HC-06 module.

Macro Definition Documentation

#define BAUD_RATE   9600

Defines speed of transmission.

Default communication baud rate in HC-06 is 9600. Others are available by changing settings in AT-mode of module.

#define BUFF_SIZE   250

Defines number of bytes in Rx and Tx buffers.

It can be any number, but too big buffers may cause memory problems.

#define OVERWRITE   1

When OVERWRITE is set and Rx buffer is full the oldest character will be replaced by receiving data.

Warning
It can be ONLY 0 or 1! OVERWRITE==1 is required when "string" functions (bt_getStr, bt_sendStr) are used. Otherwise Tx buffer lock may occur (bt_getStr uses string_count which is incremented by NULL incoming character). Functions guarantee the continuity of strings, but use it carefully. Best way is to use still same type of functions (for example bt_getChar, bt_sendChar / bt_getStr, bt_sendStr).
#define UART (   x)    XUART(x)

Preprocessor macro for automatic change of UART module.

#define UART_IRQ_PRIORITY   1

Defines how important UART interrupts will be. It can be 1, 2 or 3.

#define UART_MODULE   2

Defines which UART will be used for communication.

You can select any UART module (0-2). Default pins:

  • UART0 - PTA15(RX) PTA14(TX)
  • UART1 - PTE1 (RX) PTE0(TX)
  • UART2 - PTE17(RX) PTE16(TX)
Warning
When KL46Z is connected to ZUMO, UART 1 is useless, because on PTE0 and PTE1 is I2C bus.
#define XUART (   x)    (UART##x)

Preprocessor macro for automatic change of UART module.
